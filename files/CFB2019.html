Dimensionality Reduction using PCA

Principal Component Analysis is a useful procedure for reducing the number of predictors in the model by analyzing the input variables. In particular, when we have subsets of measurements that are highly correlated, PCA can play a key role in reducing dimensionality efficiently. In such cases, it summarizes the large number of independent variables to a few weighted linear combinations, while retaining the explanatory power.

Here, I am considering to use PCA to decompose the independent variables into an orthogonal basis (the principal components), and then select a subset of those components as variables to predict the number of wins. Generally, one selects the principal components with the highest variance that is, the components with the largest singular values because the subspace defined by these principal components captures most of the variation in the data, and thus represents a smaller space that we believe captures most of the qualities of the data. However, that standard PCA is an x-only decomposition, and sometimes lower-variance components can be critical for predicting y. Conversely, even if one components takes most of the variance, sometimes it is not important to predict y.

So, in this project, I firstly performed PCA, including y vectors, Win-Loss, and then fit a generalized linear model using the principal components.

In most of the cases, independent variables in data have different units. For example, in this study, the unit of Off Yards is yard, while the unit of Off TDs Allowed is count. Hence, the variances of Off Yards is much larger than the variances of Off TDs Allowed and variance of Off Yards dominates variance of Off TDs Allowed. A solution is to center the data at mean zero and scale it to unit standard deviation, which is easy with the scale command.

CFB.pca = prcomp(CFB.2019, center = TRUE, scale. = TRUE)

Binomial Regression

Suppose the response variable \(Y_{i}\) for \(i = 1,...,n\) is binomially distributed \(Bin(m_{i}, p_{i})\) so that:

\(P(Y_{i} = y_{i}) = \binom{m_{i}}{y_{i}}p_{i}^{y_{i}}(1-p_{i})^{m_{i}-y_{i}}\)

We further assume that the \(Y_{i}\) are independent. The individual outcomes or trials that compose the response \(Y_{i}\) are all subject to the same q predictors \((x_{i1},...,x_{iq})\).

In this study, each of the observation is representing winning records of each team. So, we can consider win as success and loss as failure in binomial distribution.

Then, we can construct a linear predictor:

\(\eta_{i} = \beta_{0} + \beta_{1}x_{i1} + ... + \beta_{q}x_{iq}\)

We can use a logistic link function \(\eta_{i} = log(p_{i}/(1-p_{i}))\)

The log-likelihood is then given by:

\(l(\beta) = \sum_{i = 1}^{n} [y_{i}\eta_{i} - m_{i}log(1+e_{i}^\eta) + log\binom{m_{i}}{y_{i}}]\)

CFB = as.data.frame(predict(CFB.pca, CFB.2019), stringsAsFactors = FALSE)
CFB$Win = as.data.frame(CFB.2019)$Win
CFB$Loss = as.data.frame(CFB.2019)$Loss

ncomp = 27

# here we will only model with the first ncomp principal components
varexpr = paste(paste("PC", 1:ncomp, sep=''), collapse='+')
fmla = paste("cbind(Win, Loss) ~", varexpr)

model = glm(fmla, family = binomial, data=CFB)

Using stepwise variable selection, we can retain only the necessary predictors in the model.

final.model = step(model, trace = FALSE)
